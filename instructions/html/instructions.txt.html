instructions

1 : Heading

A heading starts with a tab or four spaces.

1.1 : Sub Heading

A subheading starts with 2 tabs or 8 spaces.

2 : Links

If line starts with http, it will be a link

2.1 : Example

http://www.snapdeal.com

3 : How to run?

python FAQhtmlConverter.py folder_path

This will create html files for all docs in the folder.
Html files will be stored in folder_path/html

4 : What does FAQwatcher.py do?

python FAQwatcher.py folder_path

This will watch the files in the specified folder and rebuild the htmls after any change.

5 : Final steps

5.1 : Add cron to enable watcher on reboot

@reboot python /home//dev/python/FAQwatcher.py /home/krishan//docs/faq > /tmp/faqwatcher.log
Don't forget to modify the user name. :-)

5.2 : Bookmarking

In your fav browser, add the html folder (/home//office/docs/faq/html) as a bookmark for easy access to all the files.
